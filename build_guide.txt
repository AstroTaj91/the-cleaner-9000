What Changed in v2.0
The original build guide listed generic skills from skills.sh and provided basic Antigravity
prompts. This version is materially upgraded in three ways:

strategies directly into the agent’s context. The original guide had no custom skills.

or review copy.

the Review Gap tracker, and the regional segmentation display — all defined in the
custom skills.
1. Executive Summary
This document provides the exact Google Antigravity build plan to create a one-click web
application that automates Eric Velch’s GTA house cleaning playbook. The application,
named Velch Automator, features a dashboard to sync Google Business Profiles (GBPs),
analyze local competitors via the Google Places API, auto-generate SEO copy using GPT-

dispatch rules, and review copy — none of which will match Velch’s proven system. The
custom skills lock in the exact mechanisms that are documented to work.

4. The Antigravity Prompt Sequence
Execute these prompts sequentially. Always review and approve the agent’s Artifact (plan)
before allowing it to write code. Each prompt now explicitly references the relevant custom
skill.
Phase 1: Project Scaffolding & Database Schema
Prompt 1: Setup & Database
“Scaffold a new Next.js 14 App Router project with a Node.js backend and Supabase
integration. Design and implement the database schema for the ‘Velch Automator’.
Tables required: GBP_Listings (our Google profiles, including city, review_count,
google_review_link), Competitors (top 3 Maps results per city, review counts), Jobs
(bookings with status enum: pending/assigned/completed), Contractors (cleaner
network with gta_region field using values: west/north/east/central), and Customers.
Apply Supabase RLS best practices from the supabase-postgres-best-practices skill.
Create an implementation plan Artifact before writing SQL migrations.”
Phase 2: Competitor Radar & GBP Sync
Prompt 2: The Threshold Tool (Competitor Radar)
“Build the ‘Competitor Radar’ module following the velch-competitor-radar skill.
Create a backend service that takes a GTA suburb and service keyword (e.g., ‘Oakville
house cleaning’) and calls the Google Places API Text Search to fetch the top 3 ranking
businesses. Apply the Velch Threshold: flag as GO (green) if at least 2 of the top 3 have
fewer than 100 reviews, NO-GO (red) if not. Build a React component with a traffic light
visual. Generate an Artifact of the API integration plan before coding.”
Prompt 3: GBP Sync & AI Copywriter
“Build the GBP Sync & SEO Engine following the velch-gbp-ranking skill. Integrate
the Google My Business API to sync active GBP listings into the GBP_Listings table.
Build an AI Copywriter using the Vercel AI SDK and GPT-4o. The weekly post
generation function MUST use this exact prompt constraint from the skill: ‘Write a
Google Business Profile update for a house cleaning company in [City]. Naturally

Phase 3: The Review Blitz Manager
Prompt 4: Review Automation
“Build the Review Blitz Manager following the velch-review-blitz skill. Integrate

UI component showing our review count vs. the top competitor’s count per GBP listing.”
Phase 4: Mission Control & Dispatch (Human-in-the-Loop)
Prompt 5: Mission Control Interface
“Build the Mission Control screen following the velch-instant-quote skill. Integrate
Twilio Voice to route inbound calls to a web softphone in the dashboard. Implement the
exact split-panel layout from the skill: left panel is the softphone + scrollable call script,
right panel is the Instant Quote Calculator. The calculator must implement the exact
pricing matrix from the skill: 1 Bed/1 Bath = 85 wholesale, 2 Bed/1 Bath =
115 wholesale, 3 Bed/2 Bath = 155 wholesale, 4 Bed/2 Bath =
195 wholesale, Deep Clean add-on = + 35 wholesale. Display
both retail and wholesale prices side by side. Include the ‘Book Job’ form with a GTA
region selector.”
